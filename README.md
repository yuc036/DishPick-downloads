# DishPick Downloads

This repository hosts the official public download assets for DishPick.

这个仓库用于存放选菜吧 / DishPick 的官方公开下载资源。

## Includes / 内容

- Android APK release files
- Public release notes
- Stable download assets for the DishPick download page

- Android APK 安装包
- 公开发布说明
- 下载页使用的稳定下载资源

## Notes / 说明

- This repository is for distribution only.
- The main DishPick source code is maintained separately in a private repository.

- 本仓库仅用于下载分发。
- DishPick 主代码仓库独立维护，且不在此公开。

## About DishPick / 关于 DishPick

DishPick is a cross-platform cooking decision app that helps users organize recipes, plan weekly menus, and generate prep lists more easily.

选菜吧 / DishPick 是一款跨平台做饭决策应用，帮助用户整理菜谱、安排每周菜单，并更轻松地生成备菜清单。
